# Shader Study

A community-driven archive for studying and sharing shader, graphics, rendering, and game engine technologies.

Shader Study is a long-running technical study community based on `http://cafe.naver.com/shader`. For nearly two decades, we have explored a wide range of topics across shaders, graphics, rendering, and game engine development.

This repository exists to organize and share the knowledge, experiments, examples, and technical resources accumulated through our study activities. Rather than serving only as an archive, we hope it becomes an open and evolving space where practical ideas, research notes, prototypes, and implementation examples can be shared and improved over time.

## Topics

- Shader Programming
- Rendering Techniques
- Real-time Graphics
- Game Engine Architecture
- Technical Art / VFX
- Optimization
- Graphics Experiments
- Technical Documentation

## Goals

- Archive long-term study outcomes
- Share practical knowledge and research
- Publish experiments, examples, and prototypes
- Encourage open collaboration
- Build a sustainable technical resource for the community
